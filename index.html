Publish & Subscribe

Meteor servers can publish sets of documents, and clients can subscribe to those publications. Any documents the client subscribes to will be available through the find method of client collections.

Reactive Computation

These Meteor functions run your code as a reactive computation:

  • Templates
  • Tracker.autorun
  • Blaze.render and Blaze.renderWithData

Reactive Datasource

The reactive data sources that can trigger changes are:

  • Database queries on Collections
  • Session variables
  • Meteor.status
  • Meteor.user

Fine-Grained Reactivity

  • Property Selectors - Posts.find({ tag: 'jquery' }).fetch();
  • Field Selectors - Posts.find({}, {fields: {author: 1,title: 0}}).fetch();
  • Collection.cursor.count() - Posts.find({ tags: 'jquery' }).count()

Property Selectors

Deps.autorun(function(){
  var allPosts = Posts.find().fetch();
  for (var post in allPosts) {
    if (post.tag === "jquery") console.log(post.title);
  }
});
-----------------------------------
Deps.autorun(function(){
  var postsAboutKittens = Posts.find({tag:"jquery"}).fetch();
  postsAboutKittens.forEach(function (post) {
    console.log(post.title);
  });
});

Field Selectors

Deps.autorun(function(){
  Posts.find({}, {fields: {author: 1, title: 0).fetch()
    .forEach(function (post) {
      console.log(post.author);
    });
});

Collection.cursor.count()

Deps.autorun(function(){
  console.log(Posts.find({ tags: 'jquery' }).fetch().length);
});

-----------------------------------------

Deps.autorun(function(){
  console.log(Posts.find({ tags: 'jquery' }).count());
});

Ignoring Changes to Certain Reactive Values

var game = new ReactiveDict;
game.set("score", 42);
game.set("umpire", "Giraffe");

Deps.autorun(function () {
  Deps.nonreactive(function () {
    console.log("current umpire is " + game.get("umpire"));
  });
  console.log("current score is " + game.get("score") + "!");
});

Nested Autoruns

var weather = new ReactiveDict;

weather.set("sky", "sunny");
weather.set("temp", "cool");

var weatherPrinter = Deps.autorun(function () {
  console.log("The sky is " + weather.get("sky"));
  var temperaturePrinter = Deps.autorun(function () {
    console.log("The temperature is " + weather.get("temp"));
  });
});

Providers of reactive values

In the sample code, we the ReactiveDict library. A real app will use a variety of libraries that provide reactive values, like a database library that lets you do arbitrary reactive database queries (such as Meteor's Minimongo, which provides a reactive implementation of the entire MongoDB query API), or simple libraries that provide the current time or window size as a reactive value.

Consumers of reactive values
